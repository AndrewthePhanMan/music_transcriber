# Audio to MIDI Transcriber

An audio processing project that converts a WAV audio file
into a time–frequency spectrogram for automatic music transcription.

## Dependencies
- Python 3.9+
- numpy
- librosa
- matplotlib

## Installation
```bash
pip install numpy librosa matplotlib
```
